# Recipe Finder App

A modern and responsive Recipe Finder application built with HTML, CSS, and Vanilla JavaScript using TheMealDB API.

This project allows users to search for recipes, explore ingredients, and view detailed cooking instructions through an interactive and responsive user interface.

---

## Features

- Search recipes by meal name
- Fetch live recipe data from TheMealDB API
- Dynamic recipe card rendering
- Interactive recipe modal with full details
- Ingredient and measurement lists
- Error handling for invalid searches
- Responsive design for desktop and mobile
- Keyboard search support (Enter key)
- Smooth hover and modal animations
- Clean and modern UI

## How It Works

1. Users search for a recipe by name
2. The application sends a request to TheMealDB API
3. Recipe data is fetched and rendered dynamically
4. Users can open a recipe modal to view:
   - Full cooking instructions
   - Ingredients
   - Recipe details
